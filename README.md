# 📊 Sales Performance Dashboard (Power BI)

## 📌 Overview

This project presents an interactive Power BI dashboard designed to analyze sales performance and generate actionable insights.

## 📊 Key Features

* Revenue and profit analysis
* Sales trends over time
* Product-wise performance
* Region-wise sales distribution
* Contributor-level insights

## 🛠 Tools Used

* Power BI
* Excel
